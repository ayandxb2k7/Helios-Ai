## HELIOS AI 
> **Autonomous Software Engineering Platform powered by Artificial Intelligence**

Helios AI is a production-grade AI-powered software engineering platform designed to assist developers throughout the software development lifecycle. It combines intelligent code understanding, repository analysis, project management, AI-powered assistance, and modern full-stack architecture into a unified workspace.

---

## ✨ Features

- 🔐 Secure Authentication
- 🤖 AI Assistant powered by Gemini
- 📁 Repository & Project Management
- 📊 Interactive Analytics Dashboard
- 📝 AI Documentation Generation
- 🧠 Intelligent Code Understanding
- 📈 Productivity Insights
- 🌙 Dark / Light Mode
- ⚡ Modern Apple-inspired UI
- 📱 Fully Responsive Design

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/ayandxb2k7/Helios-Ai.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production

```bash
npm run build
```
